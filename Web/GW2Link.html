What is GW2Link?
It is a very simple application when gathers data such as player position, rotation, server, and map from a running Guild Wars 2 client, and sends that data to a website that requests it.

How do I use GW2Link?
Download the zip file above, and extract the exe anywhere on your computer. Run GW2Link.exe, as well as the Guild Wars 2 client, and open the map - ideally on a second monitor. You should see the download link at the top of the page change to a "Connected" status notification, and a blue icon will appear where your player is. To the right of the map you can see all of the raw data that is gathered by GW2Link.

The icon's location is a little off... what gives?
As far as I can tell, the regions specified by ArenaNet's map API are slightly off. If it isn't corrected by ArenaNet soon, I will manually fix the offsets for each map soon.

You're boring! But this is only a proof-of-concept at the moment. It serves no purpose other than to show that it's possible to display the player's location overlayed on the GW2 map. Eventually I (or someone else) plan on showing more useful information like all of the current map's running events, and events that will start soon. Won't I get banned for using mods?
There are no guarantees, but this is highly unlikely in the case of GW2Link. Due to the way the data is gathered, I don't believe GW2Link actually violates the EULA. It doesn't actually read any of GW2's memory, but rather exploits an existing feature inteded for use with the Mumble voip client - GW2Link registers a tiny block of memory on your computer, and the GW2 client periodically looks for that memory and willingly fills it with with player data.

Is this going to steal my password?
I assure you that the application does nothing more than gather small bits of data from GW2 - only data regarding the player's location - and sends it to your browser if you have loaded up the map site. Even the player data that is gathered is only used client-side to display the map - it doesn't get sent to the server.

I'm a developer - Can I use this on my GW2 map site?
Absolutely! You can use the existing GW2Link application, or download the source from GitHub, and modify it to your heart's content.
