What is GW2Link?
It is a very simple application which gathers data such as player position, rotation, server, and map from a running Guild Wars 2 client, and sends that data to a website that requests it.

How do I use GW2Link?
Download the zip file above, and extract the exe anywhere on your computer. Run GW2Link.exe, as well as the Guild Wars 2 client, and open the map - ideally on a second monitor (read on for using a tablet/phone). You should see the download link at the top of the page change to a "Connected" status notification, and a blue icon will appear where your player is. To the right of the map you can see all of the raw data that is gathered by GW2Link.

Can I view the map on my tablet/phone?
You sure can! The only requirement is that your device is using Wifi on the same local network as the PC which is running GW2 and GW2Link (the PC doesn't have to be using Wifi). All you need to do then is load up the map on your device, expand the options section in the right panel, and enter the local IP address of your PC in the "GW2Link IP" form. You can find your PC's local IP address in Windows by opening your Start menu, typing "cmd" in the search, then in the console that appears enter "ipconfig". One of the many lines that is printed out will be labled "IPv4 Address", and the listed IP should look something like "192.168.1.112".

Won't I get banned for using mods?
There are no guarantees, but this is highly unlikely in the case of GW2Link. Due to the way the data is gathered, I don't believe GW2Link violates the EULA. It doesn't actually read any of GW2's memory, but rather exploits an existing feature inteded for use with the Mumble voip client - GW2Link registers a tiny block of memory on your computer, and the GW2 client periodically looks for that memory and willingly fills it with with player data.

Is this going to steal my password?
I assure you that the application does nothing more than gather small bits of data from GW2 - only data regarding the player's location - and sends it to your browser if you have loaded up the map site. Even the player data that is gathered is only used client-side to display the map - it doesn't get sent to the server.

I'm a developer - Can I use this on my GW2 map site?
